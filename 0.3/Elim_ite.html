Module Elim_ite

module Elim_ite: sig .. end

Eliminate "if/then/else"

Example: if a&b then case else if c&d then case2 else case3 becomes (a&b) => case1; (not (a&b) & c&d) => case2; (not(a&b) & not (c&d)) => case3
